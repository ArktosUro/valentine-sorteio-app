# 💖 Valentine Lucky Slot 🎰

A fun Valentine-themed number lottery game where users enter names and numbers (1–100), then spin a casino-style reel to randomly pick a winner from a live list.

## 💘 Features

- Add participants in real time
- Assign each person a number (1–100)
- Animated spinning slot machine effect
- Slow-down casino-style reel animation
- Random winner selection from live list
- Valentine-themed UI 💖🌹

## 🎰 How it works

1. Enter a name and a number (1–100)
2. Build your live list of participants
3. Click **Spin**
4. Watch the reel animate like a casino slot machine
5. A winner is selected from the list 🎉

## 💻 Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
